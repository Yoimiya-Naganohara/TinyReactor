TinyReactor

A lightweight reactor server

Configuration

Create a config file and pass it to the server on startup.

# tinyreactor.conf
port = 8080
num_threads = 16
document_root = .
max_epoll_events = 1024
max_cache_size = 128
# custom MIME types
mime_type.webp = image/webp
mime_type.wasm = application/wasm

Run with ./TinyReactor tinyreactor.conf to load your settings
